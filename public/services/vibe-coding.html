바이브코딩 교육

코딩 경험 없는 비개발자가 AI와 함께 업무 도구를 직접 만드는 실전 트랙. 15세션 체계적 커리큘럼.

바이브코딩(Vibe Coding)은 AI에게 자연어로 의도를 전달하면 AI가 코드를 생성하고, 사람은 방향을 잡고 결과를 검증하는 새로운 개발 방식입니다. CDSA의 바이브코딩 교육은 코딩 경험이 전혀 없는 비개발자도 업무 도구를 직접 만들 수 있게 합니다.

01 / Target누구를 위한 과정인가

개발자가 없어도, 예산이 없어도, 지금 당장 필요한 도구는 내가 만들 수 있어야 합니다. CDSA의 바이브코딩 교육은 아래 네 유형의 실무자를 위해 설계했습니다.

Type 01
비개발자 실무자
Type 02
1인 기업 · 프리랜서
Type 03
기획자 · 마케터
Type 04
공무원 · 행정직

02 / Curriculum커리큘럼 구성 — 15세션

기초 웹 기술부터 최신 AI 코딩 도구까지, 단계적으로 쌓이는 구조입니다. 각 세션은 독립적으로 조합해 맞춤 설계할 수 있습니다.

세션 주제 도구
01 HTML로 만드는 첫 웹페이지 HTML / CSS
02 Python 업무자동화 I — 파일·폴더 제어 Python
03 Python 업무자동화 II — 엑셀·데이터 처리 Python / openpyxl
04 Python 업무자동화 III — 보고서·PDF 생성 Python / reportlab
05 엑셀 VBA 자동화 Excel VBA
06 PowerPoint VBA 자동화 PPT VBA
07 Google Apps Script — 스프레드시트·Gmail 자동화 Apps Script
08 Cursor로 코딩하기 — AI 페어 프로그래밍 Cursor
09 VS Code + AI 확장 환경 구성 VS Code
10 Lovable로 웹 앱 만들기 — 노코드 AI 빌더 Lovable
11 Bolt · v0 — 프론트엔드 AI 생성 Bolt / v0
12 Replit — 브라우저에서 바로 배포 Replit
13 Google AI Studio — Gemini 기반 앱 프로토타이핑 Google AI Studio
14 Claude Code — 에이전틱 AI 코딩 Claude Code
15 종합 프로젝트 — 나만의 업무 도구 완성 자유 선택

03 / Duration교육 시간 — 맞춤 설계

기관 일정과 목표 역량에 따라 세션을 선택·조합해 최소 8시간부터 24시간까지 설계합니다. 집체 교육과 온라인 병행 모두 가능합니다.

8H
입문 집중 과정
핵심 3~4세션
16H
기본 과정
도구 심화 + 실습
24H
심화 과정
전 세션 + 프로젝트

04 / Outcomes수강 후 할 수 있는 것

교육 종료 시점이 아니라, 현업에 복귀한 그 다음 날부터 바로 시작할 수 있는 것들입니다.

  • 나만의 업무 자동화 도구 제작 — 엑셀 매크로, 파일 정리 스크립트, 반복 보고서 생성기
  • 웹 앱 프로토타입 — 팀 내부용 신청·관리 웹 앱을 개발자 없이 직접 만들기
  • 데이터 대시보드 구축 — CSV·엑셀 데이터를 시각화하는 인터랙티브 대시보드
  • AI와 협업하는 개발 루틴 — 문제 정의 → 자연어 지시 → 결과 검증의 반복 사이클 체화

어떤 문제를 풀고 싶으신가요?
커리큘럼과 시간을 함께 설계합니다.
