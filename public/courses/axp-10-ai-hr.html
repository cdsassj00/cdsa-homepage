Outcomes
이 과정을 마치면
01
HR 데이터를 GPT로 분석·요약·보고
성과평가, 조직진단, 이직분석, 교육효과, 채용분석 등 인사팀 핵심 데이터를 AI 기반으로 처리합니다.
02
인사이트 도출에서 시각화까지 자동화
복잡한 HR 지표를 LLM에게 설명하고, 패턴 발견부터 차트 생성까지 하나의 흐름으로 연결합니다.
03
설문·평가 데이터에서 개선안 도출
Excel, 설문 결과, 평가표를 GPT에게 넘겨 요약, 문제점, 인사제도 개선 시사점을 자동으로 뽑아냅니다.
04
프롬프트 템플릿 → 보고서 → 슬라이드
HR 프롬프트를 체계화하고, 조직진단 보고서 자동 생성부터 PPT 슬라이드화까지 직접 경험합니다.

M1
HR 데이터 흐름과 분석 구조 이해
조직도, 성과, 이직, 교육 등 인사 데이터 흐름 파악. 항목별 해석 + GPT 설명 방식 실습. 프롬프트 구조 설계

M2

조직 KPI, 이직률, 성과평가표 구조 설명 실습. "이건 왜 중요한가요?" → GPT 해석 유도. 문서형 요약 훈련
GPT-4o, Gemini
M3
이직/퇴사 패턴 분석하기
사유 자동 분류, 고위험군 추출, 주기성 요약. GPT로 이직 위험 인사이트 도출. 보고서 문장 생성까지
Claude, GPT-4
M4
교육 효과 및 만족도 분석
교육 ROI, 만족도 조사 결과 → GPT 해석. 슬라이드용 문장 자동화. 설문결과 기반 개선 제안 요청
GPT + Excel
M5
성과평가 요약 및 개선방향 제시
다면평가 결과 → GPT 자동 문장화. 취약포인트 분류 및 추천. 피드백 리포트 자동 생성 흐름
GPT, ChatGPT
M6
채용 관련 데이터 분석 실습
지원자 통계, 전형단계 분석, 전환율 해석. GPT 개선안/패턴 도출. 채용보고서 문서화까지
GPT + Sheets
M7
인사 전략 보고서 자동 문서화
문제 정의 → 진단 → 전략 문서화. GPT 전체 구조 자동 요약 + PPT 문장 생성. 팀장 보고용 리포트
ChatGPT, Docs
M8
실전 프로젝트: HR 이슈 분석 & 전략 보고
실제 HR 문제 정의 → GPT 분석 → 인사이트 도출. 전략제안서 또는 슬라이드 보고서 완성. 발표 및 피드백
