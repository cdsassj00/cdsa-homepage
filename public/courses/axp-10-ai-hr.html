& 인사 자동화 과정

"성과에서 이직까지, GPT로 읽고 정리하는 인사 인사이트"
데이터를 설명하면, 인사 전략 보고서가 자동으로 따라온다

638,000원 VAT 포함 (공급가 580,000원 + 부가세 58,000원)

OUTCOMES학습 기대효과

  • 인사팀에서 흔히 다루는 성과평가, 조직진단, 이직분석, 교육효과, 채용분석 등 HR 데이터를 GPT 기반으로 분석·요약·보고
  • 복잡한 HR 지표와 조직 데이터를 LLM에게 설명하고 인사이트 도출 및 시각화까지 자동화
  • Excel, 설문 결과, 평가표를 기반으로 GPT에게 요약, 문제점 도출, 인사제도 개선 시사점 도출 요청 가능
  • GPT 기반으로 HR 프롬프트 템플릿화 → 조직진단 자동화 보고서 생성 → 슬라이드화까지 경험

OBJECTIVES학습목표

AI 트렌드 이해 & 업무 적용
생성형 AI의 최신 트렌드와 실무 활용 원리를 이해하고 브랜드사업본부 업무에 적용 가능한 역량 강화
교육 기획·분석 AI 접목
브랜드교육 기획, 교육자료 제작, 설문·만족도 분석, 키워드 분석 등 교육파트 주요 업무에 AI 접목
AI 실무 템플릿 도출
직무별 AI 활용 사례와 프롬프트 작성법을 학습하여 본부 내 다양한 직무에서 활용 가능한 템플릿 도출
콘텐츠 기획 속도 향상
AI 기반 자료 조사·요약·콘텐츠 제작 역량을 체화하여 교육 콘텐츠의 기획 속도와 완성도 향상

CURRICULUM · 8 MODULES커리큘럼

모듈모듈명주요 내용활용 도구
M1 HR 데이터 흐름과 분석 구조 이해 조직도, 성과, 이직, 교육 등 인사 데이터 흐름 파악. 항목별 해석 방식 + GPT 설명 방식 실습. 분석항목별 프롬프트 구조 설계 ChatGPT, GPT
M2 GPT에게 조직 데이터를 설명하는 법 조직 KPI, 이직률, 성과평가표 구조 설명 실습. "이건 왜 중요한가요?" → GPT로 해석 유도. 문서형 요약 흐름 훈련 GPT-4o, Gemini
M3 이직/퇴사 패턴 분석하기 사유 자동 분류, 고위험군 추출, 주기성 요약. GPT로 이직 위험 인사이트 도출. 보고서 문장 생성까지 실습 Claude, GPT-4
M4 교육 효과 및 만족도 분석 교육 ROI, 만족도 조사 결과 → GPT로 해석. 요약 + 슬라이드용 문장 자동화 실습. 설문결과를 기반으로 한 개선 제안 요청 GPT + Excel, ChatGPT
M5 성과평가 요약 및 개선방향 제시 다면평가 결과 요약 → GPT 자동 문장화. 취약포인트 분류 및 추천 실습. 피드백 리포트 자동 생성 흐름 학습 GPT, ChatGPT
M6 채용 관련 데이터 분석 실습 지원자 통계, 전형단계 분석, 전환율 해석 실습. GPT가 추천하는 개선안/패턴 도출 흐름. 채용보고서 문서화까지 진행 GPT + Google Sheets
M7 인사 전략 보고서 자동 문서화 문제 정의 → 진단 → 전략 흐름 문서화. GPT로 전체 구조 자동 요약 및 PPT 문장 생성. 팀장 보고용 리포트 자동 구성 실습 ChatGPT, GPT for Docs
M8 실전 프로젝트: HR 이슈 분석 & 전략 보고 실제 HR 문제 정의 → GPT 분석 → 인사이트 도출. 전략제안서 또는 슬라이드 보고서 완성. 발표 및 피드백 공유 자유도구
